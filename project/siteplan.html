Formative Math Lab – Site Plan

(This is a basic plan for my project. Details may change when I build the final website.)

Site Name

Formative Math Lab

I chose this name because the website will focus on short math questions for formative assessment. The word “Lab” suggests a place to experiment, practice, and observe how students think.

Site Purpose

The purpose of this website is to provide quick formative assessment questions for middle-grade mathematics, especially aligned with the Chilean curriculum (starting with 6th grade OA 11).

The site will:

  • Offer short questions about solving equations and other math topics.
  • Give teachers ideas for using questions as warm-ups, exit tickets, and quick checks.
  • Allow students to practice simple questions and check their understanding.

Scenarios

These are example questions a visitor might have when they visit the site:

  • “I am teaching OA 11 (solving equations). Where can I find a few short questions to check if my students understand the topic?”
  • “Can I quickly practice solving equations and see if my answers are correct?”

Color Scheme

I plan to use the following colors on my final website:

  • Dark Blue (#003366): for headings and navigation.
  • Yellow (#ffb703): for buttons and small accents.
  • Light Gray (#f7f7fb): for the page background.

Typography

I plan to use simple, easy-to-read fonts:

  • A sans-serif font (for example Arial): for headings.
  • The same sans-serif font: for body text and paragraphs.
